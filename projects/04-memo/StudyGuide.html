Checklist

이해도 체크리스트

  • Codable이 Encodable + Decodable의 조합임을 안다
  • JSONEncoder().encode()와 JSONDecoder().decode()의 타입 파라미터를 이해한다
  • didSet { save() }가 items 변경 시 자동 저장하는 원리를 설명할 수 있다
  • init() { load() }에서 저장된 데이터를 복원하는 흐름을 구현할 수 있다
  • NavigationLink(destination:)의 지연 로딩 방식을 이해한다
  • @State private var memo: Memo를 MemoEditView에서 초기화하는 패턴을 안다
Key Code

핵심 코드 패턴

1. 저장/불러오기

// MemoStore.swift @Observable class MemoStore { var items: [Memo] = [] { didSet { save() } // 변경될 때마다 자동 저장 } init() { load() } // 앱 시작 시 자동 불러오기 private func save() { let data = try? JSONEncoder().encode(items) UserDefaults.standard.set(data, forKey: "memos") } private func load() { guard let data = UserDefaults.standard.data(forKey: "memos"), let saved = try? JSONDecoder().decode([Memo].self, from: data) else { return } items = saved } }

2. MemoEditView 상태 초기화

struct MemoEditView: View { let memo: Memo @State private var title: String @State private var body: String init(memo: Memo) { self.memo = memo _title = State(initialValue: memo.title) // ← 이 패턴! _body = State(initialValue: memo.body) } }
