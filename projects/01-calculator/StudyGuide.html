Self Check

핵심 개념 체크리스트

  • @Observable 클래스와 @State 변수의 차이를 설명할 수 있다
  • enum에 연산 프로퍼티(backgroundColor, foregroundColor)를 추가하는 이유를 안다
  • GeometryReader에서 geo.size.width로 버튼 크기를 계산하는 방식을 이해한다
  • 2D 배열([[CalcButton]])을 ForEach로 중첩 순회하는 방법을 안다
  • displayText를 최대 12자리로 포맷하는 방법을 구현할 수 있다
  • 소수점 처리(0.0 → "0", 5.0 → "5")를 직접 구현할 수 있다
Code Analysis

핵심 코드 분석

CalculatorLogic의 핵심 패턴
@Observable class CalculatorLogic { var displayText = "0" private var currentValue: Double = 0 private var operator: String? = nil func tap(_ button: CalcButton) { switch button { case .digit(let d): appendDigit(d) case .operator(let op): setOperator(op) case .equals: calculate() case .clear: reset() } } }
GeometryReader 버튼 크기 계산
GeometryReader { geo in let w = geo.size.width let buttonSize = (w - 5 * 12) / 4 // 4열, 간격 12pt VStack(spacing: 12) { ForEach(buttonLayout, id: \.self) { row in HStack(spacing: 12) { ForEach(row, id: \.self) { btn in CalcButtonView(button: btn, size: buttonSize) } } } } }
