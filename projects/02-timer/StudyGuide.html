Self Check

핵심 개념 체크리스트

  • Timer.publish(every:on:in:).autoconnect()와 .cancel()의 동작을 설명할 수 있다
  • .onReceive(timer) { _ in } 클로저가 언제 호출되는지 안다
  • Circle().trim(from:0, to:)에서 progress 값 범위(0~1)를 이해한다
  • .rotationEffect(.degrees(-90))가 필요한 이유를 설명할 수 있다
  • isRunning 상태에 따라 타이머를 구독/취소하는 로직을 구현할 수 있다
  • 스톱워치에서 최소랩(green)/최대랩(red) 색상 계산 로직을 구현할 수 있다
Code Analysis

핵심 코드 분석

Timer 구독 패턴
let timer = Timer.publish(every: 1, on: .main, in: .common).autoconnect() .onReceive(timer) { _ in guard isRunning else { return } if remainingSeconds > 0 { remainingSeconds -= 1 } else { isRunning = false UINotificationFeedbackGenerator().notificationOccurred(.success) } }
