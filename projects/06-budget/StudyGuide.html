Checklist

이해도 체크리스트

  • @Model 클래스와 일반 Codable struct의 차이를 설명할 수 있다
  • .modelContainer(for: Transaction.self)가 하는 일을 이해한다
  • @Query(sort:order:)로 정렬된 데이터를 자동 로드하는 방법을 안다
  • context.insert(t)와 context.delete(t)가 즉시 반영되는 이유를 이해한다
  • #Predicate를 이용해 이번 달 데이터만 필터링하는 방법을 구현할 수 있다
  • BarMark(x:y:)로 카테고리별 지출을 차트로 그리는 방법을 안다

핵심 코드 패턴

1. SwiftData 핵심 패턴

// 모델 선언 (Transaction.swift) @Model class Transaction { var title: String var amount: Double // 수입: 양수, 지출: 음수 var category: String var date: Date var isIncome: Bool { amount > 0 } var absAmount: Double { abs(amount) } } // 앱 진입점 (BudgetApp.swift) WindowGroup { ContentView() } .modelContainer(for: Transaction.self) // DB 파일 자동 생성 // 데이터 사용 (DashboardView.swift) @Query(sort: \Transaction.date, order: .reverse) private var transactions: [Transaction]
