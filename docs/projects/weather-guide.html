Step 00
API 연동의 흐름
인터넷에서 데이터를 받아오는 전체 과정을 먼저 이해한다.
URLSession
HTTP 요청을 보내고 응답을 받는 Apple 내장 라이브러리. data(from: url) 한 줄로 GET 요청 완료.
async/await
네트워크 요청은 시간이 걸린다. async 함수를 await으로 기다리면 완료 후 다음 줄이 실행된다.
Codable 디코딩
API가 반환하는 JSON을 Swift struct로 자동 변환. CodingKeys로 snake_case → camelCase 매핑.
@MainActor
네트워크 응답은 백그라운드 스레드에서 옴. UI 업데이트는 메인 스레드에서 해야 한다.
Open-Meteo API 소개

이 프로젝트에서는 Open-Meteo (open-meteo.com) 무료 날씨 API를 사용합니다. API 키가 필요 없고, 회원가입도 불필요합니다.

// 서울 현재 날씨 요청 예시
https://api.open-meteo.com/v1/forecast
  ?latitude=37.5665
  &longitude=126.9780
  ¤t=temperature_2m,weathercode
  &timezone=Asia/Seoul
브라우저에서 먼저 테스트
위 URL을 브라우저에 붙여넣으면 JSON 응답을 바로 볼 수 있습니다. Codable 모델을 만들기 전에 응답 구조를 확인하세요.

CodingKeys — snake_case 매핑

Swift는 camelCase를 선호하지만, API는 snake_case를 씁니다. CodingKeys로 매핑합니다:

struct CurrentWeather: Codable {
    let temperature2m: Double   // Swift: camelCase
    let weathercode: Int

    enum CodingKeys: String, CodingKey {
        case temperature2m = "temperature_2m"  // JSON: snake_case
        case weathercode                         // 같으면 그냥 씀
    }
}

Step 02
WeatherService
async/await + URLSession으로 API를 호출하고, 에러를 처리한다.
전체 fetch 흐름
func fetch(city: City) async {
    isLoading = true

    // 1. URL 조립
    var components = URLComponents(string: baseURL)!
    components.queryItems = params.map { URLQueryItem(name: $0.key, value: $0.value) }
    guard let url = components.url else { return }

    do {
        // 2. 네트워크 요청 (await — 완료될 때까지 대기)
        let (data, _) = try await URLSession.shared.data(from: url)

        // 3. JSON → Swift struct
        let response = try JSONDecoder().decode(WeatherResponse.self, from: data)

        // 4. 메인 스레드에서 UI 업데이트
        await MainActor.run {
            weatherData[city.id] = response
            isLoading = false
        }
    } catch {
        await MainActor.run {
            errorMessage = error.localizedDescription
            isLoading = false
        }
    }
}
try + await — 두 가지 실패 가능성
await: 완료를 기다린다 (비동기)
try: 실패할 수 있다 (에러 던짐 가능)
네트워크 요청은 두 경우 모두 해당하므로 try await를 함께 씁니다.
뷰에서 호출 — .task
WeatherView(city: city)
    .task {
        // 뷰가 나타날 때 자동 실행, 뷰가 사라지면 자동 취소
        await service.fetch(city: city)
    }
Info.plist — App Transport Security
HTTP(비보안) URL을 사용할 경우 Info.plist에 NSAppTransportSecurity 설정이 필요합니다. Open-Meteo는 HTTPS를 사용하므로 별도 설정이 없어도 됩니다.

private var gradient: LinearGradient {
    let code = weather?.current.weathercode ?? 0
    let colors: [Color]
    switch code {
    case 0:       colors = [Color(hex: "1E90FF"), Color(hex: "87CEEB")] // 맑음
    case 51...82: colors = [Color(hex: "4A5568"), Color(hex: "718096")] // 비
    default:      colors = [Color(hex: "4A7AB5"), Color(hex: "8CA7C8")]
    }
    return LinearGradient(colors: colors, startPoint: .top, endPoint: .bottom)
}
