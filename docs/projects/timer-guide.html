Step 00
시작 전에 —
무엇을 배우는가
커리큘럼 어디와 연결되는지 먼저 파악하고 시작한다.
Timer.publish
Combine 퍼블리셔로 일정 간격마다 이벤트를 방출. .onReceive로 뷰에서 수신해 상태를 업데이트한다.
Path / Circle().trim
Circle에 trim(from:to:)을 적용해 원호를 잘라내는 방식으로 원형 프로그레스 바를 구현한다.
TabView
타이머 탭과 스톱워치 탭을 TabView로 나누고, tabItem으로 하단 탭 아이콘을 지정한다.
String(format:)
mm:ss 또는 mm:ss.xx 형식으로 시간을 포맷할 때 C 스타일 포맷 문자열을 활용한다.
선수 조건
Stage 1-1 (View 구조, VStack/HStack), Stage 1-2 (@State, Button), Stage 1-4 (Sheet, NavigationStack) 을 먼저 완료해야 합니다.
완성 앱 미리보기

iOS 기본 시계 앱의 타이머·스톱워치 탭과 동일한 동작을 구현합니다:

  • 타이머: 시/분/초 선택 Picker → 카운트다운 → 완료 알림
  • 원형 링 애니메이션으로 남은 시간 시각화
  • 스톱워치: 시작/정지/랩/초기화 버튼
  • 랩 목록에서 최단 랩은 초록, 최장 랩은 빨강으로 강조
  • 100분의 1초 단위(mm:ss.xx) 실시간 업데이트
공부하는 방법
완성 코드를 먼저 실행해서 동작을 확인한 뒤, 빈 프로젝트에서 이 가이드를 따라 직접 처음부터 만들어보세요. 막히면 완성 코드를 참고하되, 복붙은 피하고 직접 타이핑하는 것이 훨씬 효과적입니다.
Step 01
앱 구조 설계
TabView로 화면을 나누고 파일 구조를 먼저 잡는다.
화면 분해

앱은 하나의 TabView 아래 두 탭으로 구성됩니다:

①
타이머 탭 (TimerView)
원형 링 + 남은 시간 텍스트 + 시작/취소 버튼. 탭하면 TimePickerSheet가 올라온다.
②

TabView vs NavigationStack
TabView는 탭 간에 독립적인 상태를 유지합니다. 타이머가 실행 중일 때 스톱워치 탭으로 이동해도 타이머는 계속 돌아갑니다. 각 뷰가 각자의 ViewModel 인스턴스를 @State로 소유하기 때문입니다.
Step 02
원형 프로그레스 바
Circle().trim과 StrokeStyle로 링 애니메이션을 만든다.
핵심 원리

SwiftUI의 Circle()에 .trim(from: 0, to: progress)를 적용하면 원의 일부만 그려집니다. progress가 0이면 아무것도 없고, 1이면 완전한 원입니다.

기본적으로 3시 방향(오른쪽)이 시작점이므로, .rotationEffect(.degrees(-90))로 12시 방향으로 돌려줍니다.

RingProgressView 전체 코드
struct RingProgressView: View {
    var progress: Double   // 0.0 ~ 1.0
    var lineWidth: CGFloat = 12
    var color: Color = .orange

    var body: some View {
        ZStack {
            // 뒤 트랙 (회색 원)
            Circle()
                .stroke(Color.gray.opacity(0.2), lineWidth: lineWidth)

            // 앞 링 (진행률)
            Circle()
                .trim(from: 0, to: progress)
                .stroke(color,
                    style: StrokeStyle(
                        lineWidth: lineWidth,
                        lineCap: .round   // 끝을 둥글게
                    )
                )
                .rotationEffect(.degrees(-90))
                .animation(.linear(duration: 1), value: progress)
        }
    }
}
TimerView에서 progress 계산
// 경과 시간 / 전체 설정 시간 = 진행률
private var progress: Double {
    guard vm.totalSeconds > 0 else { return 0 }
    return Double(vm.remainingSeconds) / Double(vm.totalSeconds)
}

RingProgressView(progress: progress)
    .frame(width: 260, height: 260)
lineCap: .round
StrokeStyle에 lineCap: .round를 주면 진행 링의 앞끝이 둥글게 마무리되어 iOS 시계 앱과 같은 느낌을 줍니다. .butt (기본값)는 각지게 잘립니다.
셀프 체크
Q1
progress를 1.0에서 0.0으로 줄이면 링이 시계 방향으로 줄어드는가?
Q2

Q3

Step 03
타이머 화면
Timer.publish로 1초마다 카운트다운하고 완료 시 알림을 보낸다.
Timer.publish 선언

Combine의 Timer 퍼블리셔를 사용합니다. autoconnect()를 붙이면 구독 즉시 타이머가 시작됩니다:

let timer = Timer
    .publish(every: 1, on: .main, in: .common)
    .autoconnect()
TimerViewModel
@Observable
class TimerViewModel {
    var totalSeconds: Int = 0
    var remainingSeconds: Int = 0
    var isRunning: Bool = false

    func start() {
        guard remainingSeconds > 0 else { return }
        isRunning = true
    }

    func tick() {
        guard isRunning, remainingSeconds > 0 else { return }
        remainingSeconds -= 1
        if remainingSeconds == 0 { finish() }
    }

    func finish() {
        isRunning = false
        UINotificationFeedbackGenerator()
            .notificationOccurred(.success)
    }

    func cancel() {
        isRunning = false
        remainingSeconds = totalSeconds
    }
}
TimerView에서 tick 수신
RingProgressView(progress: progress)
    .onReceive(timer) { _ in
        vm.tick()
    }
TimePickerSheet — 시/분/초 설정
struct TimePickerSheet: View {
    @Binding var hours: Int
    @Binding var minutes: Int
    @Binding var seconds: Int

    var body: some View {
        HStack {
            Picker("시간", selection: $hours) {
                ForEach(0..<24) { Text("\($0)시").tag($0) }
            }.pickerStyle(.wheel)

            Picker("분", selection: $minutes) {
                ForEach(0..<60) { Text("\($0)분").tag($0) }
            }.pickerStyle(.wheel)

            Picker("초", selection: $seconds) {
                ForEach(0..<60) { Text("\($0)초").tag($0) }
            }.pickerStyle(.wheel)
        }
    }
}
타이머를 끄는 방법
.onReceive는 뷰가 살아있는 한 항상 이벤트를 받습니다. isRunning 플래그로 tick() 내부에서 가드하거나, Cancellable을 저장해 명시적으로 cancel()을 호출하는 방법을 씁니다.
Step 04
스톱워치 화면
랩 기록 배열과 100분의 1초 포맷을 구현한다.
StopwatchViewModel
@Observable
class StopwatchViewModel {
    var elapsed: Double = 0    // 총 경과 초 (소수 포함)
    var laps: [Double] = []    // 랩 기록 배열
    var isRunning = false
    private var lapStart: Double = 0

    func tick(_ interval: Double = 0.01) {
        guard isRunning else { return }
        elapsed += interval
    }

    func lap() {
        let lapTime = elapsed - lapStart
        laps.insert(lapTime, at: 0)   // 최신 랩이 맨 위
        lapStart = elapsed
    }

    func reset() {
        elapsed = 0; laps = []; lapStart = 0; isRunning = false
    }
}
mm:ss.xx 포맷 함수
func formatTime(_ t: Double) -> String {
    let m  = Int(t) / 60
    let s  = Int(t) % 60
    let cs = Int((t.truncatingRemainder(dividingBy: 1)) * 100)
    return String(format: "%02d:%02d.%02d", m, s, cs)
}
최단/최장 랩 색상 강조
private var minLap: Double? { laps.min() }
private var maxLap: Double? { laps.max() }

func lapColor(_ lap: Double) -> Color {
    if laps.count < 2 { return .primary }   // 랩이 1개면 강조 없음
    if lap == minLap { return .green }
    if lap == maxLap { return .red }
    return .primary
}
랩 List 렌더링
List(Array(vm.laps.enumerated()), id: \.offset) { i, lap in
    HStack {
        Text("랩 \(vm.laps.count - i)")
            .foregroundStyle(lapColor(lap))
        Spacer()
        Text(formatTime(lap))
            .font(.system(.body, design: .monospaced))
            .foregroundStyle(lapColor(lap))
    }
}
100분의 1초 타이머 성능
0.01초 간격 Timer는 뷰 업데이트가 매우 잦습니다. 메인 디스플레이 텍스트처럼 꼭 필요한 부분만 업데이트되도록 뷰를 작게 분리하면 성능에 유리합니다. CADisplayLink를 쓰는 것도 방법입니다.
