Prerequisites

🔧 Getting Started

Two things you need before starting — a Mac and Xcode. Swift comes bundled inside Xcode.

Language

What is Swift?

A programming language created by Apple. Used to build iPhone, iPad, and Mac apps.

Swift was introduced by Apple in 2014, designed to replace the previously used Objective-C. It is now the standard language for iOS/macOS/watchOS/visionOS app development.

Why Swift

  • Safe — The Optional system catches null crashes at compile time
  • Fast — Near C++ performance. Designed around value types (structs)
  • Readable — No semicolons, type inference, trailing closures, and other concise syntax
  • Perfect with SwiftUI — SwiftUI is Swift-exclusive. Declarative UI written in Swift syntax
Note

Swift is open source. It can run on Linux and Windows, but to build iOS apps you must have a Mac + Xcode.

Tool

What is Xcode?

Apple's development tool. Code editing, building, simulator, and app distribution — all in one place.

Xcode is an IDE (Integrated Development Environment). You can write code, build apps, run them in the simulator, and submit to the App Store — all with Xcode alone.

What's inside Xcode

  • Code Editor — Auto-completion, syntax highlighting, error display, Fix-it suggestions
  • Swift Compiler — Swift is included without any separate installation
  • iOS Simulator — Test your app on iPhone/iPad screens without a physical device
  • SwiftUI Preview (Canvas) — See UI changes in real time without building
  • Interface Builder — Drag & drop UI editing (rarely used with SwiftUI)
  • Instruments — Performance and memory profiling tools
  • Signing & Distribution — Sign your app and upload to App Store Connect
Requirements

macOS only. Xcode is not available on Windows or Linux. If you don't have a Mac, a used MacBook Air (M1 or later) is the most practical option.

Install

Installing Xcode

Free download from the Mac App Store. It's a large download, so allow some time.

1
Search for "Xcode" in the Mac App Store
Open the App Store on your Mac and type "Xcode" in the search bar. Look for the blue icon with the Apple developer logo.
2
Click "Get" → Install
It's free. However, it's about 12–15 GB in size, so make sure you have enough disk space. The download may take 30 minutes to over an hour.
3
First launch → Install additional components
When you first open Xcode, an "Install Additional Components" popup will appear. Click Install to set up the simulator and other required tools.
4
Sign in with Apple ID (optional)
Go to Xcode → Settings → Accounts and add your Apple ID. This gives you a free developer account to run apps on a physical device. If you're only using the simulator, you can do this later.
Tip

Instead of the App Store, you can download directly from developer.apple.com/xcode to choose a specific version. This is useful when you need a beta version or when the App Store is slow.

Verify

Verify Installation

The quickest way to confirm everything is set up correctly.

1
Open Xcode and click "Create New Project"
Select "Create New Project..." from the Welcome screen.
2
Select iOS → App → Next
Choose iOS for the platform and App for the template, then click Next.
3
Enter a project name → Confirm Interface: SwiftUI
Type any name (e.g., "TestApp") in the Product Name field. Make sure Interface is set to SwiftUI, then click Next → choose a save location.
4
Click the ▶ Run button (or ⌘R)
Press the ▶ button in the top-left corner or use the ⌘R shortcut. If the simulator launches and you see "Hello, world!" — you're all set.
Success

If you can see "Hello, world!" in the simulator, Xcode + Swift + SwiftUI are all working correctly. Now let's start Challenge 1-1.
